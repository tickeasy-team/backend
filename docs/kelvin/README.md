# Tickeasy 票務系統後端開發筆記

## 專案概述
這是一個完整的企業級票務管理系統後端專案，使用 Node.js + Express + TypeScript + PostgreSQL 技術棧開發。

## 技術架構
- **後端框架**: Node.js + Express.js
- **程式語言**: TypeScript
- **資料庫**: PostgreSQL + Supabase
- **ORM**: TypeORM
- **身份驗證**: JWT + Passport.js
- **部署**: Docker + Docker Compose
- **AI功能**: OpenAI API
